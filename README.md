# SQL Data Analysis Specialization

Welcome to my SQL learning journey! This repository documents my step-by-step progress from a SQL beginner to an advanced Data Analyst using PostgreSQL. 

## Objective
To build a comprehensive portfolio of SQL scripts and data analysis projects, demonstrating real-world database management, data querying, and business problem-solving skills.

## 📂 Repository Structure 
This repository is organized into five progressive phases, covering everything from basic syntax to advanced analytical concepts.

### `01_Basic_Setup/`
* Database creation, table structures (`CREATE TABLE`), data types, and basic data insertion (`INSERT`).

### `02_Core_Analyst_Toolkit/`
* **Filtering & Sorting:** `WHERE`, `AND/OR/NOT`, `IN`, `BETWEEN`, `LIKE`, and `ORDER BY`.
* **Data Manipulation:** Removing duplicates with `DISTINCT` and using Math/Arithmetic functions.
* **Aggregation:** `GROUP BY`, aggregate functions (`SUM`, `AVG`, `COUNT`), and post-aggregation filtering with `HAVING`.

### `03_Relational_Data/`
* **Table Joins:** `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, and `Self-Joins`.
* **Stacking Data:** Combining result sets using `UNION`.

### `04_Advanced_Analysis/`
* **Logic & Cleaning:** Conditional logic (`CASE` statements) and String/Date Functions.
* **Complex Queries:** Subqueries and Common Table Expressions (CTEs).
* **Window Functions:** Analytics over partitions (`ROW_NUMBER`, `RANK`, `LAG`, `LEAD`, `SUM OVER`).
* **Theory & Formatting:** Data `Pivoting` and understanding the SQL Execution Order.

### `05_Capstone_Project/`
* **The Business Problem:** Identifying the single most valuable "VIP" customer in every product category to launch a targeted rewards program.
* **End-to-End Analytics:** A comprehensive business report combining CTEs, Joins, Aggregations, and Window Functions to solve the business problem and extract actionable insights.
