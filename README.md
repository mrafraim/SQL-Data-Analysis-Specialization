# SQL-Data-Analysis-Specialization

Welcome to the SQL learning journey! This repository documents step-by-step progress from a SQL beginner to an advanced Data Analyst using PostgreSQL. 

## Objective
To build a comprehensive portfolio of SQL scripts and data analysis projects, demonstrating real-world database management, data querying, and business problem-solving skills.

## Learning Roadmap
This repository is organized into five core phases based on real-world data analyst requirements:

* **[Phase 1] Foundations & Setup:** Database creation, table structures (`CREATE TABLE`), data types, and basic data insertion (`INSERT`).
* **[Phase 2] The Core Analyst Toolkit:** Filtering (`WHERE`), Grouping (`GROUP BY`), Aggregation (`SUM`, `AVG`, `COUNT`), and sorting (`ORDER BY`).
* **[Phase 3] Relational Data:** Connecting data using Inner/Left/Right Joins, Unions, and mapping table relationships.
* **[Phase 4] Advanced Analysis:** Subqueries, Common Table Expressions (CTEs), and Window Functions.
* **[Phase 5] The Capstone Project:** End-to-end data cleaning, complex business reporting, and query optimization.

## Repository Structure
Each folder represents a specific step in my learning process, containing `.sql` files with the queries and logic used for that lesson.

* `01_Basic_Setup/`: Contains the initial database setup and mock e-commerce dataset creation.
